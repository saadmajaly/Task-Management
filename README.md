<h1> Summary </h1>

<p>Developed a comprehensive web application using the Django framework, integrating user authentication and dynamic front-end components using HTML, CSS, and JS for an enhanced user experience. This project not only allows users to sign up, log in securely, and perform CRUD operations on tasks but also incorporates client-side validation using JavaScript to ensure a smooth and responsive user interface. </p>
